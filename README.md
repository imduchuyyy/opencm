## Business model:
- Free: Free bot, 1000 messages per month, 10 messages per minute, no custom prompts, no knowledge upload, no image input, no link input.
- Pro: 49$/month or 490$/year, 10,000 messages per month, 100 messages per minute, custom prompts, knowledge upload (< 5Mb), image input, link input.
- Max: 99$/month or 990$/year, 50,000 messages per month, 100 messages per minute, custom prompts, knowledge upload (< 20Mb), image input, link input.
- Custom: Custom pricing, unlimited messages, custom prompts, knowledge upload, image input, link input.
